# Sound Examples

Sound examples for the nonlinear transverse string experiment are included.

| Folder | Description |
| -------- | ------- |
| `sav_88200Hz_2sec_75modes_02b814394ad8deb51762b7949e8fb3c8` | Training dataset |
| `sav_96000Hz_3sec_75modes_45f296f20960bb05ae8150b2eab8cf81` | Validation dataset |
| `sav_96000Hz_3sec_75modes_827174835397d508a1a39775b6959823` | Test dataset |

Each folder contains audio for:
1. Target nonlinear trajectory of a string `#.wav`.
2. Predicted nonlinear trajectory of a string `#_pred.wav`.
3. Linear trajectory of a string `#_lin.wav`.
